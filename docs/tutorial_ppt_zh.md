## 3. 产品定位与核心特性

- 轻量、离线、本地运行
- 字符级精准 ASR（Paraformer），时间线对齐友好（字幕需要二次生成）
- 一键去除静音/口头禅（“水词”）
- 生成SRT，按区间生成成片，编码可选 NVENC/拷贝/重编码

---

## 4. 系统与环境要求

- 操作系统：Windows（推荐 Win10）
- Python：3.11（需用户自行预装）
- 项目提供一键脚本创建虚拟环境以及基本依赖
- 可选 GPU：CUDA 11.8 + cuDNN 8.9.x（需用户自行预装）
- FFmpeg：需用户自备并放入 `third_party/ffmpeg/bin`
- 可选ImDisk：（需用户自行预装）

---

## 5. 准备工作（依 README）

- 安装 Python 3.11（保持 py launcher 或 Add to PATH）
- 下载免安装版FFmpeg解压 ，将 `bin` 放到 `third_party/ffmpeg/bin/`
- 可选：安装 CUDA 11.8 与 cuDNN 8.9.x
- 可选：下载安装ImDisk，将安装路径加入系统环境path
- 在项目根目录执行：

```bash
install.bat
```

- 运行 Web 界面：

```bash
run_webapp.bat
```

- 首次执行视频转写时会自动下载离线模型

## 7. 界面布局速览

- 侧边栏：工程文件操作、转写与剪辑入口、导出工具、选项设置
- 主工作区：
  - 视频预览（播放/比例/适配）
  - 文稿编辑（分段文本、删除/恢复、搜索、常用水词设置）
  - 时间线（缩放、自适应、区间可视）

## 8. 新建与导入

- 打开工程文件或新建项目
- 在“导入音视频”处选择源媒体（仅测试过mp4，其他格式需自行测试）
- 选择识别模型与运行设备后，点击“转写”

---

## 9. 识别引擎与模型

- 识别引擎：Paraformer
- 模型：`paraformer-zh`（中文）、`paraformer-en`（英文）
- 运行设备：`自动`/`CUDA`/`CPU`（推荐 CUDA 有显卡时）

---

## 10. 一键转写

- 点击“转写”后开始 ASR 转写
- 转写完成后，文字与时间戳将出现在“文稿编辑”区域

---

## 11. 文稿编辑（基础）

- 一键删除无声片段（静音）
- 一键去除水词（口头禅）：在输入框填写“呃/呀/就是/其实”等，用空格分隔
- 支持搜索并删除单个或全部匹配结果
- 勾选“显示删除内容”可临时查看已删除文本

---

## 12. 常用水词管理

- 打开“常用水词列表”，维护全局通用的水词词表
- 保存后在所有文稿中复用
- 词条以空格、顿号或换行分隔

---

## 13. 时间线与可视化

- 缩小/放大/自适应 3 种视图控制
- 结合文稿编辑的删除操作，时间线会显示保留/删除的区段
- 播放器不可预览最终效果，仅能播放原视频供参考

---

## 14. 导出字幕与视频

- 导出 SRT：一键生成字幕文件（含时间轴）注意：不建议基于编辑后的文稿直接导出SRT字幕，因为文稿一旦被编辑，字幕跟音轨很难同步。只有把生成的视频重新导入再次转写为文稿，此时导出的SRT字幕才能完美契合语音片段。如果只是为了生成字幕而使用本软件，并不需要对文稿进行任何编辑行为，可以在完成转写后直接导出SRT，此时字幕与音轨完全同步。
- 生成剪辑：按“保留区间”输出成片
- 输出文件名可自定义（留空则使用项目 ID）

---

## 15. 编码与音频选项

- 重新编码策略：
  - `自动选择`/`直接拷贝`/`NVIDIA NVENC`/`强制重新编码`
- 零交叉吸附：`启用/禁用`（减少音频切割爆音）
- 音频交叉淡化（ms）：平滑拼接过渡
- 剪辑批次大小：控制分段处理规模（提升长视频稳定性）

---

## 16. 工程与版本管理

- 工程：打开/保存/另存/关闭
- 版本：另存为版本，支持历史版本列表回看与恢复
- 建议关键节点前保存快照，便于回溯

---

## 17. 常见问题排查
- `ffmpeg not found`：检查 `third_party/ffmpeg/bin/ffmpeg.exe` 是否存在
- CUDA 未检测到：核对 CUDA/cuDNN 安装与环境变量
- 首次运行模型加载慢：正常现象，后续会缓存
- 输出不同步/卡顿：尝试 NVENC、开启零交叉吸附、调整批次大小

---
## 18. 快捷键
- 文稿编辑时可以配合ctrl,shift加鼠标左键对文稿字符进行连选和加选。选中后字符会标红，标红状态的字符可以按del键删除。
- 被删除的字符可以按ctrl+z返回
- 所有被删除的字符可以通过文稿编辑区右下角的“显示删除内容”选项来一键显示。
- 在显示已删除内容状态下，鼠标再次点击被删除的字符，对应的字符会取消标红恢复为未删除状态。
- “一键删除无声片段”和“一键去除水词”，这两个按钮只有当目标存在时才会生效，会在鼠标悬停时标红所有目标，会在点击按钮时删除所有目标。
- “一键去除水词”右侧文本输入框可以临时定义水词，最右侧设置可以开启水词列表供用户登记永久过滤的水词。
- “删除查找内容”文本输入框可以作为水词之外的另一种补充，因为它对于输入的文本进行过滤并且支持单次删除和全部删除，当前文稿显示区会聚焦显示首个查找结果，单击删除后会聚焦到下一个查找结果，便于用户对每个查找结果进行人工判定。
- 当鼠标悬停在文稿任意字符上时按空格键可以在时间轴上跳转到对应的时间切片。
- 鼠标双击时间轴上的任意切片可以使文稿显示区聚焦到该切片对应的字符。

## 19. 目录结构与数据位置

```text
subtitle-cut/
├─ models/              离线模型
├─ src/                 核心源代码（web app / cli）
├─ third_party/         第三方依赖路径占位（FFmpeg 放置处）
├─ data/                Web App 运行数据（项目、导出、日志 等）
├─ install.bat          虚拟环境初始化脚本
├─ run_webapp.bat       启动 Web 界面脚本
└─ docs/                文档与素材
```
